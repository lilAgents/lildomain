# lilDomain

Available Domain Finder

Find short, available domain names right now. lilDomain generates keyword mixes and pronounceable coinages, then confirms each one is unregistered with a live DNS and RDAP check.

Part of [Hot App Summer](https://hotappsummer.netlify.app) by [lilAgents](https://lilagents.com): one small, useful web tool shipped every day through June.

## Run it locally

This is an [Astro](https://astro.build) site.

```bash
pnpm install
pnpm dev
```

To build for production:

```bash
pnpm build
```
